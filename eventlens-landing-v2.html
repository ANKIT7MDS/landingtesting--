FAQ
Questions photographers ask before switching.
Short answers. No marketing fluff.
Do guests need to install an app?

No. Guests can use a normal browser link / QR code. (You can add an optional app later, but keep the default frictionless.)

Will guests see other people’s photos?

EventLens is privacy-first: the guest flow is designed to show only matched photos. Add an optional PIN for extra control.

Can clients select photos for editing or album?

Yes. Guests can mark favorites. You get a clean selection list without endless WhatsApp screenshots.

What about pricing and limits?

Plans are in ₹, designed for Indian studios. You can enforce storage/search limits per photographer account.

Can I add my studio branding?

Yes — logo, colors, and studio name on the gallery pages. You can also add per-event branding for corporate clients.

Do you compress images?

You control the upload quality. For best experience, offer both “fast preview” and “original download” options as needed.
